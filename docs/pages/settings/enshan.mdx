### 配置说明

|        参数         |              说明               |
| :-----------------: | :-----------------------------: |
| _**ENSHAN**_.cookie | 恩山无线论坛 帐号的 cookie 信息 |

<Steps>

### 打开官网

进入[恩山无线论坛](https://www.right.com.cn/forum/)

### 打开开发者工具

按 `F12` 打开开发者工具，刷新页面

### 点击 Network 标签

### 选择 Doc 标签

### 选中 www.right.com.cn

### 下滑找到 cookie 全选复制即可
