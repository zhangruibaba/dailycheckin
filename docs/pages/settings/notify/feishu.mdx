# 飞书

### 配置示例

```json filename="config.json" copy
{
  "FSKEY": "",
  "MERGE_PUSH": ""
}
```

|       参数       |                                           说明                                            |
| :--------------: | :---------------------------------------------------------------------------------------: |
|   _**FSKEY**_    | `https://open.feishu.cn/open-apis/bot/v2/hook/xxxxxx` **xxxxxx** 部分就是需要填写的 FSKEY |
| _**MERGE_PUSH**_ |                       **true**: 将推送消息合并；**false**: 分开推送                       |
